## Installation

### Prerequisites

- Docker
- Nginx
- Portainer ( Optional )

### Installation Steps

1. Clone the repository and submodules
2. Change desired env variable within `./build.sh & ./docker-compose.yml`
3. Build the mf app to docker image locally: `./build.sh`
4. Served in desired URL
